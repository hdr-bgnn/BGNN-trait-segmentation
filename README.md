# BGNN-trait-segmentation
--------------------------

PyTorch implementation of Fish trait segmentation model.

### Requirements:
------------------
- Python (>=3.6)
- PyTorch
- **segmentation_models_pytorch**
- **albumentations**
- Numpy
- CV2
- Matplotlib
- Pillow
- Seaborn
- Pylab
- Pandas

### Source File
----------------
Run the **single_model_training.ipynb** notebook for training.
